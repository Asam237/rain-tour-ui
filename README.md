## Rain Tour UI

Rain Tour UI Frontend
<br/>

## Stack

- Typescript
- Next.js 13
- Tailwind CSS
- Deployed on Vercel
  <br/>

## Installation

```bash
git clone https://github.com/Asam237/rain-tour-ui.git

cd rain-tour-ui

yarn install

yarn dev
```
